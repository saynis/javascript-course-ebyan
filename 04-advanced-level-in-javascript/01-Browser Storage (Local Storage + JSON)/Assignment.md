# DOM + Local Storage Project: Student Marks Tracker

## Objective
Create a simple web app that stores student marks using DOM and Local Storage.

## Requirements

Create three files:

- index.html
- style.css
- script.js

## Features

1. Create a form with:
   - Student Name
   - Subject
   - Mark
   - Submit button

2. When the user submits the form:
   - Display the student data on the page
   - Save the data in Local Storage

3. When the page is refreshed:
   - The saved students must still appear





## Local Storage Key
Use this key:

studentsMarks

## Bonus Challenge
Add simple colors:
- Green for Passed
- Red for Failed
