## Task 5 — Arrow Function ✍️

Rewrite this function as an arrow function:

```js
function square(n){
   return n * n
}

function getFullName(firstName, lastName) {
  return firstName + " " + lastName;
}
```

Then test it with:

```js
console.log(square(5))

```
